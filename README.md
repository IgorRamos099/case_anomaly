# Análise de Anomalias em Dados de Marketing

## Contexto
Analisar um dataset de marketing digital e encontrar 4 anomalias ocultas:

1. Um canal de mídia com ROI excessivamente baixo
2. Um canal de mídia com um ROI absurdamente alto
3. Um mês específico que foi muito abaixo da média em vendas
4. Um dia atípico que apresentou um pico de vendas isolado

## Anomalias encontradas
1. Canal com ROI mais baixo : Meta Ads (ROI médio: 1.49)
2. Canal com ROI absurdamente alto : TikTok Ads (ROI médio: 28.10)
3. Mês abaixo da média : Setembro/2025 (R$ 4.444.201,96)
4. Pico isolado de vendas : 19/01/2026 (R$ 453.772,19)

## Tecnologias Utilizadas
- **Python 3**
- **Pandas**
